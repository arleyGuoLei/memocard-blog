    # 人教版高中 (必修一) 词汇列表

    单词备注释义其它
    新媒体的基本特征数字化、网络、互动性
    数字时代影响“把关”的变化原因：从信息相对稀缺到信息超载、平台媒体的兴起、用户信息消费习惯的改变
    大众传媒的新闻价值“把关标准”：时效性、重要性、趣味性、接近性
    算法的主要功能信息排序、分类、联合、过滤
    “策展”的主体有记者/编辑策展、个人策展、策略策展、算法策展
    自动化新闻的应用和推广，将在微观和宏观层面产生潜在影响微观方面：对记者的影响、对用户的影响
    宏观方面：对媒介组织的影响、对社会的影响
    媒介效果是指媒介对用户多方面的影响，主要有认知 情感 态度 行为
    自动化新闻在初期算法程序完成后，不经人为干预，由软件或算法自动生成新闻故事的过程。
    多种结构性的变化都会带来相应的伦理影响。结构性包括经济结构 政治结构 叙事结构 关系结构
    新闻分发的主要模式有编辑分发、社交分发、算法分发
    二级传播信息流经由大众媒体到意见领袖，再到普通受众的传播过程。
    社会网络的主要特征是开放性 动态性 群体性 以自我为中心
    社交分发用户通过社交媒体来获取信息的一种模式，信息主要沿着社交关系网络流动和扩散。
    新媒体时代，传播环境呈现出鲜明的特征碎片化 个人化 分众化
    叙事结构变化带来的伦理影响传统新闻遵循的规范是准确、平衡、中立、发布前的核实；互联网新闻则注重及时、互动性、透明性、发布后的修正
    “5W”传播模式谁、说了什么、通过什么渠道、对谁说、产生了什么效果来介绍有关传播者、新闻生产、传播渠道、受众研究和效果研究这五大领域在新媒体环境下的新闻发展。
    自动化新闻的工作原理标准化五环节：收集数据、找出有趣的事件、选取角度、生成新闻故事、发布新闻故事
    VR技术对媒介伦理的影响A、感官刺激的伦理风险 B、人为建构对公众可能具有更大的操纵性和欺骗性 C、强大的沉浸感和互动性容易刺激非理性消费，导致用户与现实社会的疏离
    伦理推导的四个步骤波特方格伦理推导四步骤：陈述事实、分析价值观、运用伦理原则、厘清忠实对象
    平台媒体既拥有媒体的专业编辑权威性，又拥有面向用户平台所特有的开放性的数字内容实体。
    公共领域的主要组成要素有公众 公众舆论 公众场所和媒介
    新媒体在中国的三个发展阶段及特征web1.0时代“人与信息的连接”【内容网络】，web2.0时代“人与人的连接”【关系网络】，web3.0时代“人与物的连接”【服务网络】。
    数据新闻的生产流程数据采集、数据处理、数据可视化、生成新闻故事
    大众媒介时代的“多层次”把关模式媒介人员工作流程层面的把关；媒体定位及影响范围的组织层面把关；媒介外社会团体层面的把关；社会系统层面的把关等。
    新媒体概念基于数字技术、网络技术及其他现代信息技术或通信技术的具有高度互动性的媒介形态，包括网络媒体、手机媒体和这两者融合形成的移动互联网，以及其他数字媒体形式。
    公众及其特征关注普遍利益而不受国家和任何其他权力影响并拥有理性思考能力的群体，他们自愿组合在一起形成公众。具有三个特征——拥有共同关注的普遍利益、自愿性组合、拥有一定的规模。
    算法分发的局限性A、算法分发重‘迎合’轻‘引导’，削弱新闻业的公共服务属性；
    B、 关注当下目标实现，未把长期目标纳入考量范畴；
C、 分发过程可能存在着更加隐蔽的偏见、歧视。 |    |
